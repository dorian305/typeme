TypeMe.js

How to use

Find an element to animate in: let myElement = document.querySelector("myElement");

Define an array of strings: let myStringArray = ["String 1", "String 2", ...];

> If it's only 1 string, you can define it as it is, no need to put it in array: let myString = "My string";

Call the animation: let myAnimation = new typeMe({ element: myElement, strings: myStringArray });

let elem = document.querySelector("myElement"); let str = ["String 1", "String 2", ...]; let myAnimation = new typeMe({ element: myElement, strings: myStringArray, //Extend with options if needed });

Options

typingSpeed: int - the speed (in milliseconds) at which the animation will animate letter typing.

deletingSpeed: int - the speed (in milliseconds) at which the animation will animate letter backspacing.

stringBackspaceDelay: int - the time (in miliseconds) before typed out string will start backspace animation.

stringTypeDelay: int - the time (in miliseconds) before the new string will start typing animation.

animationStartDelay: int - the time (in miliseconds) before the animation starts.

loop: bool - if set to true, animation will repeat indefinitely or until animation instance is destroyed.

> stopOnLastString flag must be disabled to allow animation looping.

stopOnLastString: bool - if set to true, the last string will not be erased.

destroyVerticalLine: bool - if set to true, the vertical line will be deleted when animation finishes.

onAnimationComplete: function - executes a function on animation completion.

let myAnimation = new typeMe({ element: myElement, strings: myStringArray, //Default values typingSpeed: 70, deletingSpeed: 30, stringBackspaceDelay: 1000, stringTypeDelay: 0, animationStartDelay: 0, loop: false, stopOnLastString: false, destroyVerticalLine: false, onAnimationComplete: () => { } });

How to install

Import stylesheet: <link href="css/typeMe.css" rel="stylesheet" type="text/css">

Import javascript: <script src="js/typeMe-min.js"></script>

Make sure the paths to the files are valid.

Methods

Methods accept callback functions.

.toggle(function); - toggle animation's pause/resume states

.restart(function); - restarts the animation

.destroy(function); - destroy animation instance
